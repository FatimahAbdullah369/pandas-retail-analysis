# Retail Sales Analysis using Pandas

A beginner-friendly data analysis project demonstrating core data manipulation techniques in Python using the `pandas` library. This project builds upon foundational concepts learned during the Google Data Analytics Professional Certificate.

## Business Scenario
This project analyzes a mock retail dataset containing transaction records across multiple product categories (Electronics, Home, Apparel) to answer basic business questions regarding sales volume and revenue generation.

## Key Data Skills Demonstrated
- **Data Ingestion**: Loading and parsing structured data into a pandas DataFrame.
- **Feature Engineering**: Creating a calculated `Total_Revenue` column combining price and volume.
- **Data Filtering**: Applying boolean indexing to isolate specific segments (e.g., separating Electronic goods).
- **Aggregation & Grouping**: Leveraging `.groupby()` to calculate total revenue, transaction volume, and average prices by category.

## Tech Stack
- **Language**: Python 3
- **Libraries**: Pandas
- **Environment**: Jupyter Notebook (Anaconda)

## Key Insights Found
- Identified revenue-generating performance across product categories.
- Evaluated transactional volume trends to discover consumer purchase habits.
